Jill had been with the same web host for years when they announced they were shutting down. Her site was built on an outdated CMS that no one supported anymore — moving it wasn't straightforward.

We migrated the entire site to Hugo, converting every page and image. The result loads in under a second, is hosted for free on GitHub Pages, and Jill can now update her own content directly. No more hosting bills, no more platform dependencies.

This is what a site rescue looks like: everything you had, working better, costing nothing to run.
